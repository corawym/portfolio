What is echo

First for freelancers to send feedback form to their peers/clients and receive opinions from them

  • Create a feedback form
  • Set up different types of questions
  • Send out the survey
  • Create a project
  • Collect the feedback result

Registration

Using Firebase's authentication and user management systems with their name, email and password

Create a survey

First naming the form, setting up questions, and then sending out by emails

Add a project

Allowing user to create a project if it is a project based feedback

Process

It worked in a group of 4 developers and developed based on the UX/UI designers' vision. I am one of the app developer. I created components built using valid JSX and included React Redux for state management. We used Git for version control and team collaboration.
