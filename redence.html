What is REDen:ce

It is a web application which allows teachers of RED Academy the ability to mark their students' attendance. It also allows the students to be able to check their attendance and to get an idea on where they stand on their attendance.

Before this app, the way teachers mark their students' attendance was using an excel sheet and the students would have to ask their teachers directly to find out their own attendance. All that hassle will be removed by this app.

Teacher dashboard

Teachers are able to mark their student's attendance, add students in the teacher dashboard and response to student's message.

Student dashboard

Students are able to check their attendance, and send messages to the respective teachers.

Login

Process

I created this real-time JavaScript web application with a group of three developers in 2 weeks. We designed the interface and built using Meteor as an application platform, with React as a rendering library and MongoDB as a database.
